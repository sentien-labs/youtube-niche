# Code of Conduct

This project is for people who want to build more transparent tooling around YouTube niche
research. Keep discussions useful, specific, and respectful.

## Expected Behavior

- Be kind and direct.
- Critique ideas, code, data, and assumptions rather than people.
- Explain tradeoffs when you disagree.
- Assume contributors may have different experience levels with YouTube, data analysis, or Python.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing another person's private information.
- Spam, self-promotion unrelated to the project, or misleading claims about results.
- Intentionally submitting secrets, private API data, or non-public user data.

## Reporting

If something needs attention, open a private security advisory when it involves secrets or data
safety, or contact the maintainers through the GitHub repository. Maintainers may remove comments,
close issues, or block accounts that make the project less safe or useful for contributors.
